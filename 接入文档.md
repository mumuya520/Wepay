## 🚀 快速开始

### 1. 获取商户信息

登录商户后台（http://localhost:3006/merchant），获取以下信息：
- **商户号 (mch_no)**: 例如 `M515637`
- **商户 ID (mch_id)**: 例如 `3`
- **API Key**: 在系统设置中配置 `simple_order_api_key`

### 2. 配置 API Key

管理员在后台设置中配置：
```sql
INSERT INTO setting (key, value) VALUES ('simple_order_api_key', '你的密钥');
```

或通过管理后台界面配置。

## 🔌 API 接口

### 创建订单

**接口地址**: `POST http://localhost:3006/api/simple/create`

**认证方式**: 
- Header: `X-API-Key: 你的API密钥`
- 或 Query: `?api_key=你的API密钥`

**请求参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| mch_id | string | 是 | 商户号或商户ID（例如：M515637 或 3） |
| out_trade_no | string | 是 | 商户订单号（唯一，建议格式：网站前缀+时间戳+随机数） |
| amount | string | 是 | 订单金额（单位：元，例如：100.00） |
| subject | string | 是 | 订单标题（例如：购买VIP会员） |
| pay_type | string | 是 | 支付方式（alipay/wxpay/qqpay 等） |
| notify_url | string | 否 | 异步通知地址（支付成功后回调） |
| return_url | string | 否 | 同步跳转地址（支付完成后跳转） |

**响应示例**:

```json
{
  "code": 0,
  "msg": "success",
  "data": {
    "order_id": "W20260530123456789",
    "mch_id": "M515637",
    "out_trade_no": "ORDER20260530001",
    "amount": "100.00",
    "subject": "购买VIP会员",
    "pay_type": "alipay",
    "qr_code": "https://qr.alipay.com/xxx",
    "pay_url": "https://pay.alipay.com/xxx"
  }
}
```

**错误码**:

| code | 说明 |
|------|------|
| -401 | API Key 无效或未配置 |
| -1 | 参数格式错误或参数不完整 |
| -2 | 商户不存在或已禁用 |
| -5 | 金额错误 |
| -6 | 商户订单号已存在 |
| -7 | 暂无可用支付通道 |
| -8 | 订单创建失败 |
| -9 | 支付通道错误 |

## 💻 接入示例

### PHP 示例

```php
<?php
// WePay 支付接入示例

class WePay {
    private $apiUrl = 'http://localhost:3006/api/simple/create';
    private $apiKey = '你的API密钥';
    private $mchId = 'M515637';  // 你的商户号
    
    /**
     * 创建支付订单
     */
    public function createOrder($params) {
        $data = [
            'mch_id' => $this->mchId,
            'out_trade_no' => $params['out_trade_no'],
            'amount' => $params['amount'],
            'subject' => $params['subject'],
            'pay_type' => $params['pay_type'] ?? 'alipay',
            'notify_url' => $params['notify_url'] ?? '',
            'return_url' => $params['return_url'] ?? ''
        ];
        
        $ch = curl_init();
        curl_setopt($ch, CURLOPT_URL, $this->apiUrl);
        curl_setopt($ch, CURLOPT_POST, true);
        curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($data));
        curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
        curl_setopt($ch, CURLOPT_HTTPHEADER, [
            'Content-Type: application/json',
            'X-API-Key: ' . $this->apiKey
        ]);
        
        $response = curl_exec($ch);
        curl_close($ch);
        
        return json_decode($response, true);
    }
}

// 使用示例
$wepay = new WePay();

$result = $wepay->createOrder([
    'out_trade_no' => 'ORDER' . time() . rand(1000, 9999),
    'amount' => '100.00',
    'subject' => '购买VIP会员',
    'pay_type' => 'alipay',
    'notify_url' => 'https://你的网站.com/notify',
    'return_url' => 'https://你的网站.com/success'
]);

if ($result['code'] == 0) {
    // 成功，跳转到支付页面
    $payUrl = $result['data']['pay_url'];
    header('Location: ' . $payUrl);
} else {
    // 失败
    echo '下单失败：' . $result['msg'];
}
?>
```

### Python 示例

```python
import requests
import time
import random

class WePay:
    def __init__(self, api_key, mch_id):
        self.api_url = 'http://localhost:3006/api/simple/create'
        self.api_key = api_key
        self.mch_id = mch_id
    
    def create_order(self, params):
        """创建支付订单"""
        data = {
            'mch_id': self.mch_id,
            'out_trade_no': params['out_trade_no'],
            'amount': params['amount'],
            'subject': params['subject'],
            'pay_type': params.get('pay_type', 'alipay'),
            'notify_url': params.get('notify_url', ''),
            'return_url': params.get('return_url', '')
        }
        
        headers = {
            'Content-Type': 'application/json',
            'X-API-Key': self.api_key
        }
        
        response = requests.post(self.api_url, json=data, headers=headers)
        return response.json()

# 使用示例
wepay = WePay(
    api_key='你的API密钥',
    mch_id='M515637'
)

result = wepay.create_order({
    'out_trade_no': f'ORDER{int(time.time())}{random.randint(1000, 9999)}',
    'amount': '100.00',
    'subject': '购买VIP会员',
    'pay_type': 'alipay',
    'notify_url': 'https://你的网站.com/notify',
    'return_url': 'https://你的网站.com/success'
})

if result['code'] == 0:
    print(f"下单成功！支付链接: {result['data']['pay_url']}")
    print(f"订单号: {result['data']['order_id']}")
else:
    print(f"下单失败: {result['msg']}")
```

### JavaScript (Node.js) 示例

```javascript
const axios = require('axios');

class WePay {
    constructor(apiKey, mchId) {
        this.apiUrl = 'http://localhost:3006/api/simple/create';
        this.apiKey = apiKey;
        this.mchId = mchId;
    }
    
    async createOrder(params) {
        const data = {
            mch_id: this.mchId,
            out_trade_no: params.out_trade_no,
            amount: params.amount,
            subject: params.subject,
            pay_type: params.pay_type || 'alipay',
            notify_url: params.notify_url || '',
            return_url: params.return_url || ''
        };
        
        try {
            const response = await axios.post(this.apiUrl, data, {
                headers: {
                    'Content-Type': 'application/json',
                    'X-API-Key': this.apiKey
                }
            });
            return response.data;
        } catch (error) {
            return { code: -1, msg: error.message };
        }
    }
}

// 使用示例
(async () => {
    const wepay = new WePay('你的API密钥', 'M515637');
    
    const result = await wepay.createOrder({
        out_trade_no: `ORDER${Date.now()}${Math.floor(Math.random() * 10000)}`,
        amount: '100.00',
        subject: '购买VIP会员',
        pay_type: 'alipay',
        notify_url: 'https://你的网站.com/notify',
        return_url: 'https://你的网站.com/success'
    });
    
    if (result.code === 0) {
        console.log('下单成功！支付链接:', result.data.pay_url);
        console.log('订单号:', result.data.order_id);
    } else {
        console.log('下单失败:', result.msg);
    }
})();
```

支付成功后，系统会向你的 `notify_url` 发送 POST 请求：

**通知参数**:

```json
{
  "order_id": "W20260530123456789",
  "out_trade_no": "ORDER20260530001",
  "mch_id": "M515637",
  "amount": "100.00",
  "real_amount": "100.00",
  "pay_type": "alipay",
  "state": 1,
  "pay_time": 1780105482,
  "sign": "签名字符串"
}
```

```php
<?php

// ...

    $orderId = $data['out_trade_no'];
    $amount = $data['amount'];

    // updateOrderStatus($orderId, 'paid');

```

### 同步跳转 (return_url)

用户支付完成后，会跳转到你的 `return_url`，携带订单信息。

## ❓ 常见问题

### 1. 如何获取 API Key？

管理员在后台执行：
```sql
INSERT INTO setting (key, value) VALUES ('simple_order_api_key', 'your-secret-key-here');
```

### 2. 支持哪些支付方式？

- `alipay` - 支付宝
- `wxpay` - 微信支付
- `qqpay` - QQ钱包
- 其他通道根据配置而定

### 3. 订单号重复怎么办？

确保 `out_trade_no` 唯一，建议格式：
```
网站标识 + 时间戳 + 随机数
例如：MYSITE20260530123456789
```

### 4. 如何测试？

1. 使用 Postman 或 curl 测试接口
2. 查看商户后台的订单列表
3. 检查支付通道配置是否正确

### 5. 生产环境部署注意事项

- ✅ 修改 API URL 为生产域名
- ✅ 使用强密码作为 API Key
- ✅ 配置 HTTPS
- ✅ 设置正确的 notify_url 和 return_url
- ✅ 做好日志记录和监控
