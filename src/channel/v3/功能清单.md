## 🚀 快速开始

```bash
# 1. 编译
cd src/channel/v3
mkdir build && cd build
cmake ..
make

# 2. 初始化数据库
psql -U postgres -f ../database_postgresql.sql

# 3. 修改配置
vim ../config.yaml

# 4. 运行
./wepay_v3_server
```

## 📝 注意事项

1. **数据库**: 使用PostgreSQL而非MySQL
2. **邮件服务**: EmailService.cpp需要补充完整实现
3. **Sentinel**: 限流熔断预留了接口，需要集成Sentinel C++ SDK
4. **监控**: 性能监控和链路追踪需要集成OpenTelemetry
5. **测试**: 建议补充单元测试和集成测试
